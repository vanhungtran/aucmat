# (Deprecated) AUC table for each numeric predictor column

`tableroc()` is deprecated. Use
[`aucmat()`](https://vanhungtran.github.io/aucmat/reference/aucmat.md)
for matrix screening. Internal imputation is no longer supported; impute
your data before passing it to
[`aucmat()`](https://vanhungtran.github.io/aucmat/reference/aucmat.md).

## Usage

``` r
tableroc(X, y, ...)
```
