\begin{abstract}
\textbf{Summary:} The area under the receiver operating characteristic curve
(AUC) is the standard metric for evaluating continuous biomarkers against
binary outcomes. When screening thousands of molecular biomarkers, researchers
need matrix-wide AUC estimation with proper inference, multiplicity adjustment,
and stability assessment. We present \texttt{aucmat}, an R package providing a
matrix-first architecture: \texttt{aucmat(X, y)} computes direction-preserving
AUCs with DeLong or stratified-bootstrap confidence intervals for every column
of a numeric matrix in a single call. The package supports Benjamini-Hochberg
multiplicity adjustment, three hypothesis-testing frameworks (superiority,
non-inferiority, equivalence via TOST), an omnibus Wald test for equality of
multiple correlated AUCs via the joint DeLong covariance matrix, and bootstrap
rank-stability analysis. A two-stage Hurdle-AUC model handles zero-inflated
data (scRNA-seq, microbiome). Eight simulation generators produce correlated
biomarkers with controlled properties under exchangeable, AR(1), block, or
user-supplied correlation structures with built-in calibration diagnostics.
Over a dozen visualization functions are provided. The package is freely
available under the MIT license at \url{https://github.com/vanhungtran/aucmat}.

\textbf{Availability:} \texttt{aucmat} v0.2.0, R $\geq$ 4.0.0, MIT license.
\url{https://github.com/vanhungtran/aucmat} (source),
\url{https://vanhungtran.github.io/aucmat} (documentation).\\
\textbf{Contact:} lucas.tran@ck-care.ch
\end{abstract}

\section{Introduction}
% ===========================================================================

Receiver operating characteristic (ROC) analysis is the dominant framework
for evaluating continuous biomarkers against binary outcomes in diagnostic
medicine, prognostic modeling, and molecular screening
\citep{Hanley1982,DeLong1988,Pepe2004}. The area under the ROC curve (AUC)
summarises the probability that a randomly chosen positive subject has a
higher biomarker value than a randomly chosen negative subject.

Modern molecular profiling technologies routinely measure thousands of
features on a few hundred subjects. A proteomics panel may contain 5,000
proteins; a transcriptomics experiment 20,000 genes. The core task --
identifying which biomarkers discriminate cases from controls -- requires
computing and testing an AUC for each feature, adjusting for multiplicity,
comparing top candidates, and assessing ranking stability. Yet no single
widely-used R package integrates these steps.

The R ecosystem provides excellent point solutions. \texttt{pROC}
\citep{Robin2011} is the gold standard: it computes DeLong confidence
intervals using the Sun \& Xu (2014) $\mathcal{O}(N\log N)$ algorithm
\citep{Sun2014}, tests paired or unpaired AUC differences via
\texttt{roc.test()}, provides partial AUC with bootstrap inference, binormal
and density-smoothed ROC curves, and multiclass AUC via Hand \& Till
\citep{Hand2001}. However, pROC operates one biomarker at a time.
\texttt{ROCR} \citep{Sing2005} offers 28 performance measures and visualisation
but no statistical inference. \texttt{precrec} \citep{Saito2017} adds fast
C++-based precision-recall curves and confidence bands but provides no DeLong
inference or multiplicity adjustment. \texttt{colAUC} \citep{Tuszynski2024}
computes column-wise AUCs efficiently but provides no inference whatsoever.
\texttt{dtComb} \citep{Yerlitas2025} combines two biomarkers via 142 methods
but does not perform matrix-wide screening. \texttt{cancerclass}
\citep{Budczies2024} builds classifiers from high-dimensional data but lacks
DeLong-based screening.

\texttt{aucmat} fills this gap with a \textit{matrix-first} design. The
primary function \texttt{aucmat(X, y)} accepts $\mathbf{X} \in
\mathbb{R}^{n \times p}$ and a binary outcome $\mathbf{y} \in \{0,1\}^n$,
returning point estimates, standard errors, confidence intervals, raw and
multiplicity-adjusted $p$-values, and diagnostic metadata for every biomarker
in one call. Downstream functions for comparison, stability, cross-validation,
panel construction, and power analysis consume the same S3 object, preserving
outcome encoding throughout.

\section{Methods}
% ===========================================================================

\subsection{Direction-preserving AUC}

For biomarker $X$ and binary outcome $Y \in \{0,1\}$, the AUC equals the
Mann-Whitney U probability:
\begin{equation}
\text{AUC} = P(X_i > X_j \mid Y_i = 1, Y_j = 0) \label{eq:auc}
\end{equation}
Most software reports $\max(\text{AUC}, 1 - \text{AUC})$, silently reversing
the biomarker. \texttt{aucmat} preserves the observed direction, reporting
$\text{AUC}_{\text{raw}}$ (may be $< 0.5$), $\text{AUC}_{\text{strength}} =
0.5 + |\text{AUC}_{\text{raw}} - 0.5|$ (for ranking), and
\texttt{effect\_direction}. A tumour suppressor with $\text{AUC}_{\text{raw}}
= 0.15$ discriminates as strongly as an oncogene with 0.85.

\subsection{DeLong variance and joint covariance}

The DeLong estimator \citep{DeLong1988,Sun2014} uses placement values:
\begin{align}
V_{10}(X_i) &= \frac{1}{n_0} \sum_{j:Y_j=0}
  \bigl[ \mathbf{1}(X_j < X_i) + \tfrac{1}{2}\mathbf{1}(X_j = X_i) \bigr],
  \quad i \in \mathcal{I}_1 \label{eq:V10} \\
V_{01}(X_j) &= \frac{1}{n_1} \sum_{i:Y_i=1}
  \bigl[ \mathbf{1}(X_i > X_j) + \tfrac{1}{2}\mathbf{1}(X_i = X_j) \bigr],
  \quad j \in \mathcal{I}_0 \label{eq:V01}
\end{align}
Let $S_{10} = (n_1-1)^{-1}\sum_{i\in\mathcal{I}_1}(V_{10}(X_i)-\bar{V}_{10})^2$
and $S_{01} = (n_0-1)^{-1}\sum_{j\in\mathcal{I}_0}(V_{01}(X_j)-\bar{V}_{01})^2$.
The variance estimator is:
\begin{equation}
\widehat{\text{Var}}(\widehat{\text{AUC}}) = \frac{S_{10}}{n_1} + \frac{S_{01}}{n_0}
  \label{eq:delong_var}
\end{equation}

For $p$ biomarkers on common subjects, the $p \times p$ joint covariance is:
\begin{equation}
\widehat{\mathbf{\Sigma}} = \frac{1}{n_1}\widehat{\text{Cov}}(\mathbf{V}_{10})
  + \frac{1}{n_0}\widehat{\text{Cov}}(\mathbf{V}_{01}) \label{eq:joint_cov}
\end{equation}
This supports an omnibus Wald test of $H_0: \text{AUC}_1 = \cdots =
\text{AUC}_p$. With contrast matrix $\mathbf{C}$ (default: comparing
biomarkers $2,\ldots,p$ against biomarker 1):
\begin{equation}
W = (\mathbf{C}\hat{\boldsymbol{\theta}})^\top
  (\mathbf{C}\widehat{\mathbf{\Sigma}}\mathbf{C}^\top)^{-1}
  (\mathbf{C}\hat{\boldsymbol{\theta}}) \sim \chi^2_{r} \label{eq:wald}
\end{equation}
using the effective rank $r$ via eigenvalue tolerance. At $p=2$, $W$ equals
the squared paired-DeLong $z$-statistic within numerical tolerance.

\subsection{Multiplicity and stability}

Screening $p$ biomarkers produces $p$ tests. \texttt{aucmat} applies
Benjamini-Hochberg \citep{Benjamini1995}, Holm, or Bonferroni correction,
reporting adjusted $p$-values as $q$-values. For TOST equivalence
\citep{Schuirmann1987}, $p_{\text{equiv}} = \max(p_{\text{lower}},
p_{\text{upper}})$ with adjustment on this combined value.

Bootstrap rank-stability (\texttt{auc\_stability()}) resamples data $B$ times
(stratified by outcome), recomputes the full screening, and reports median/IQR
rank distributions, top-$k$ selection probabilities, and pairwise co-selection
frequencies \citep{Efron1994}.

\subsection{Hurdle-AUC for zero-inflated data}

Standard AUC assumes continuous biomarkers. On zero-inflated data (scRNA-seq,
microbiome), 60\%+ of values are zero, and standard AUC degrades to near 0.5.
The Hurdle-AUC models zeros and expression as separate processes:
\begin{align}
\text{Stage 1 (zero hurdle):}\quad
  P(X_j = 0 \mid Y) &= \text{logistic}(\beta_0 + \beta_1 Y) \label{eq:hurdle1} \\
\text{Stage 2 (magnitude):}\quad
  \text{AUC}(X_j \mid X_j > 0) &\text{ computed on expressed values only}
  \label{eq:hurdle2}
\end{align}
The composite score for observation $i$ is:
\begin{equation}
S_i = \begin{cases}
  \hat{p}_i & \text{if } X_{ij} = 0 \\
  0.5\,\hat{p}_i + 0.5\,\tilde{x}_{ij} & \text{if } X_{ij} > 0
\end{cases} \label{eq:hurdle_score}
\end{equation}
where $\hat{p}_i$ is the predicted non-zero probability from Stage~1 and
$\tilde{x}_{ij}$ is the normalised biomarker value.

\subsection{Multivariate simulation engine}
\label{sec:simulation}

The simulation engine is a core contribution of \texttt{aucmat}. We describe
it in full mathematical detail, as it underpins the package's reproducibility
guarantees and distinguishes it from ad-hoc simulation approaches.

\subsubsection{The equal-variance binormal model}

Under the standard binormal model \citep{Hanley1982}, for biomarker
$k \in \{1,\ldots,p\}$:
\begin{equation}
X_k \mid Y = 0 \sim \mathcal{N}(\mu_{k0}, \sigma^2), \qquad
X_k \mid Y = 1 \sim \mathcal{N}(\mu_{k1}, \sigma^2) \label{eq:binormal}
\end{equation}
The standardised mean separation $\delta_k = (\mu_{k1} - \mu_{k0})/\sigma$
determines the AUC via:
\begin{equation}
\text{AUC}_k = \Phi\!\left(\frac{\delta_k}{\sqrt{2}}\right)
  \label{eq:auc_delta}
\end{equation}
where $\Phi(\cdot)$ is the standard normal CDF. Inverting gives the
closed-form calibration used by all \texttt{aucmat} simulators:
\begin{equation}
\delta_k = \sqrt{2} \cdot \Phi^{-1}(\text{AUC}_k) \label{eq:delta_cal}
\end{equation}
For $\text{AUC}=0.8$, $\delta \approx 1.19$; for $\text{AUC}=0.9$, $\delta
\approx 1.81$. The class means are centred so the $n$-weighted grand mean is
zero:
\begin{equation}
\mu_{k0} = \delta_k \cdot \left(-\frac{n_1}{n}\right), \qquad
\mu_{k1} = \delta_k \cdot \left(\frac{n_0}{n}\right) \label{eq:centered_means}
\end{equation}

\subsubsection{Class-conditional multivariate normal (\texttt{simulate\_auc\_matrix})}

For $p$ biomarkers with correlation matrix $\mathbf{R}$, the class-conditional
distribution generalises Eq.~\eqref{eq:binormal} to the multivariate case:
\begin{equation}
\mathbf{X} \mid Y = 0 \sim \mathcal{N}_p(\boldsymbol{\mu}_0, \mathbf{R}), \qquad
\mathbf{X} \mid Y = 1 \sim \mathcal{N}_p(\boldsymbol{\mu}_1, \mathbf{R})
  \label{eq:mvn_sim}
\end{equation}
where $\boldsymbol{\mu}_0 = (\mu_{10},\ldots,\mu_{p0})^\top$ and
$\boldsymbol{\mu}_1 = (\mu_{11},\ldots,\mu_{p1})^\top$ follow
Eq.~\eqref{eq:centered_means}. Both classes share the same $\mathbf{R}$.

The outcome $Y$ is fixed \textit{before} any biomarker is drawn. In
\texttt{outcome\_mode = "exact"}, $n_1 = \lfloor n\pi \rfloor$ observations
are assigned $Y=1$ with randomised order; in \texttt{"bernoulli"}, each
$Y_i \sim \text{Bernoulli}(\pi)$. When $Y$ is user-supplied, its exact values
and row order are preserved.

The matrix $\mathbf{R}$ is built from a named structure:

\begin{itemize}[nosep]
  \item \textbf{exchangeable}: $R_{ij} = \rho$ for all $i \neq j$.
    Requires $\rho > -1/(p-1)$ for positive definiteness.
  \item \textbf{AR(1)}: $R_{ij} = \rho^{|i-j|}$.
  \item \textbf{block}: $R_{ij} = \rho_w$ if $i,j$ in same block,
    $\rho_b$ otherwise. Block sizes must sum to $p$.
  \item \textbf{user}: $R_{ij}$ directly from the supplied matrix.
\end{itemize}

When $\mathbf{R}$ is not positive definite, \texttt{feasibility = "nearest"}
projects to the closest valid correlation matrix via eigenvalue clipping
followed by re-normalisation to unit diagonal. The adjustment is reported in
\texttt{\$feasibility} with Frobenius norm and maximum element-wise change.

After generation, each biomarker is standardised to zero mean and unit
variance (an affine transform that preserves both AUC and Pearson correlation).
Achieved correlation is computed on within-class centred residuals to avoid
mixture-induced inflation (Simpson's paradox).

\subsubsection{Gaussian copula with perturbation (\texttt{simulate\_auc\_copula})}

This simulator separates correlation control from discriminative signal:

\textbf{Phase~1 -- Gaussian Copula.} For each class independently, $n_c$
draws are taken from $\mathcal{N}_{p+1}(\mathbf{0}, \mathbf{R}_{\text{work}})$,
where $\mathbf{R}_{\text{work}}$ is a $(p+1)\times(p+1)$ matrix combining the
biomarker correlation $\mathbf{R}$ (upper-left $p\times p$) with latent
outcome correlations $\boldsymbol{\rho}$ (last row/column). The latent
correlations are approximated from target AUCs:
\begin{equation}
\rho_k^{\text{latent}} = \frac{\delta_k}{\sqrt{1 + \delta_k^2}}
  \label{eq:latent_rho}
\end{equation}
Each column is rank-transformed to uniform quantiles, then probit-transformed
to standard normal scores. This copula step preserves rank correlations
exactly regardless of marginal distributions.

\textbf{Phase~2 -- Iterative Perturbation.} After Phase~1, each biomarker's
empirical AUC, $\widehat{\text{AUC}}_k$, is compared to its target. The gap
$\Delta_k = \text{AUC}_k - \widehat{\text{AUC}}_k$ is translated to a mean
shift via the binormal calibration:
\begin{equation}
\Delta\delta_k = \sqrt{2}\left[
  \Phi^{-1}(\text{AUC}_k) - \Phi^{-1}(\widehat{\text{AUC}}_k)
\right] \label{eq:perturb}
\end{equation}
The shift $\Delta\delta_k \cdot \sigma_k \cdot \eta^t$ is applied
class-conditionally (positive class receives $+(n_0/n)$ times the shift,
negative class $-(n_1/n)$), where $\eta \in (0,1)$ is the step decay and $t$
the iteration index. Perturbation stops when $|\Delta_k| < \tau$ for all $k$ or
after $T_{\max}$ iterations.

\subsubsection{Hurdle simulator (\texttt{simulate\_hurdle\_auc})}

Generates zero-inflated biomarkers via a two-stage process. For biomarker $j$:

\textbf{Stage~1 -- Zero hurdle.} Logistic parameters are solved from
user-specified per-class zero rates:
\begin{align}
\beta_0 &= \text{logit}(p_0), \quad
  p_0 = P(X_j = 0 \mid Y = 0) \label{eq:hurdle_beta0} \\
\beta_1 &= \text{logit}(p_1) - \text{logit}(p_0), \quad
  p_1 = P(X_j = 0 \mid Y = 1) \label{eq:hurdle_beta1}
\end{align}
For each subject, $Z_{ij} \sim \text{Bernoulli}(\text{logistic}(\beta_0 +
\beta_1 Y_i))$ determines whether $X_{ij} = 0$.

\textbf{Stage~2 -- Magnitude.} For observations with $Z_{ij} = 0$
(expressed), values are drawn from a log-normal distribution with
class-conditional means calibrated to achieve the target non-zero AUC via
Eq.~\eqref{eq:delta_cal}:
\begin{equation}
X_{ij} \mid X_{ij} > 0, Y_i = c \sim
  \text{LogNormal}\!\left(\mu_c, \sigma^2\right) \label{eq:lognormal_hurdle}
\end{equation}
where $\mu_1 - \mu_0$ is set by the binormal shift. Values are clipped to
$[0, \infty)$.

\subsubsection{Other simulators}

\texttt{generate\_data\_probit()} uses a latent probit formulation where
$\mathbf{X}$ and a latent $Y^* \sim \mathcal{N}(0,1)$ are jointly
$\mathcal{N}_{p+1}(\mathbf{0}, \mathbf{\Sigma})$, with $Y =
\mathbf{1}\{Y^* > \Phi^{-1}(1-\pi)\}$. The full $(p+1)\times(p+1)$
correlation matrix simultaneously controls both biomarker-biomarker and
biomarker-outcome associations. The latent correlations $\rho_k =
\text{Cor}(X_k, Y^*)$ are calibrated numerically via bisection over a large
calibration sample ($n_{\text{cal}} = 100{,}000$).

\texttt{generate\_data\_analytical()} constructs biomarkers sequentially from
highest to lowest AUC. For $k \geq 2$, $X_k$ is built as a linear combination
of $X_1,\ldots,X_{k-1}$ (weighted by $\mathbf{R}_{k,1:k-1}
\mathbf{R}_{1:k-1,1:k-1}^{-1}$ to achieve target correlations) plus an
independent residual carrying the remaining mean separation. This is fast
(closed-form, no calibration loop) but links AUC and correlation.

\texttt{generate\_auc\_vector()} constructs a single score with exact empirical
AUC via rank assignment. Given $n_1$ positives and $n_0$ negatives, achievable
AUCs form a discrete grid with step $\Delta_{\text{AUC}} = 1/(n_1 n_0)$. The
required rank sum is solved and normal quantile scores are assigned.

\texttt{generate\_auc\_cor\_vector()} extends this with a strictly increasing
Box-Cox transform tuned to achieve a target Pearson correlation without
changing the AUC (since monotone transforms preserve ranks).

\texttt{simulate\_auc\_correlation()} repeatedly draws from the binormal model
and records achieved (AUC, $r$) pairs to characterise their sampling
distribution.

\subsubsection{Simulation validation}

\texttt{validate\_simulation()} repeats a specification across $R$ independent
replicates with deterministic seed streams ($\text{seed} + r - 1$). For each
replicate it records achieved AUCs and correlations, then reports:
\begin{align}
\text{Bias}_k &= \frac{1}{R}\sum_{r=1}^{R}
  (\widehat{\text{AUC}}_{kr} - \text{AUC}_k) \label{eq:bias} \\
\text{RMSE}_k &= \sqrt{\frac{1}{R}\sum_{r=1}^{R}
  (\widehat{\text{AUC}}_{kr} - \text{AUC}_k)^2} \label{eq:rmse} \\
\text{MCSE}_k &= \frac{\text{SD}_k}{\sqrt{R}} \label{eq:mcse}
\end{align}
plus target-interval hit rates for both AUCs and correlations. A bias large
relative to MCSE signals a systematic calibration error.

\section{Implementation}
% ===========================================================================

\texttt{aucmat} is implemented in pure R (6,200 lines, 34 source files).
The primary function performs six steps: (1)~input validation, (2)~outcome
normalisation (accepting logical, 0/1, factor, or character), (3)~matrix-wide
AUC via Mann-Whitney rank formulation, (4)~DeLong or bootstrap inference,
(5)~multiplicity adjustment, (6)~assembly with ranking and warnings.

All functions return S3 objects with \texttt{print()}, \texttt{summary()},
\texttt{plot()}, and \texttt{as.data.frame()} methods. Structured error
classes allow programmatic handling. An RNG-safety mechanism
(\texttt{with\_seed()}) restores \texttt{.Random.seed} on exit.

Dependencies: pROC ($\geq$ 1.18), mvtnorm, ggplot2, rlang. Optional
Suggests: GGally, glmnet, matrixStats, knitr, rmarkdown, ggrepel.

\section{Usage}
% ===========================================================================

Basic screening:
\begin{verbatim}
fit <- aucmat(X, y, ci = "delong", adjust = "BH")
print(fit); summary(fit)
plot_auc_rank(fit, n_label = 4)
\end{verbatim}

Paired comparisons with hypothesis testing:
\begin{verbatim}
compare_auc(fit, X, y, reference = "X1")  # superiority
compare_auc(fit, X, y, reference = "X1",
  hypothesis = "noninferiority", margin = 0.05)
compare_auc(fit, X, y, reference = "X1",
  hypothesis = "equivalence", margin = 0.15)
compare_auc_global(fit, X, y)  # H0: all AUCs equal
\end{verbatim}

Hurdle-AUC for zero-inflated data:
\begin{verbatim}
sim <- simulate_hurdle_auc(n = 400, prevalence = 0.3,
  target_hurdle_aucs = c(0.85, 0.72),
  zero_rate_neg = c(0.55, 0.80),
  zero_rate_pos = c(0.25, 0.70))
fit <- hurdle_auc(as.matrix(sim$data[,1:2]), sim$data$truth)
# Standard AUC ~0.53 -> Hurdle AUC ~0.90
\end{verbatim}

Cross-validation, panel construction, power analysis:
\begin{verbatim}
cv <- cv_aucmat(X, y, n_folds = 5, seed = 42)
panel <- fit_auc_panel(X[,top4], y, method = "ridge")
power_auc_matrix(target_aucs = 0.70, power = 0.80,
  prevalence = 0.3)
\end{verbatim}

Five capabilities are unique to \texttt{aucmat}: (i)~matrix-first screening
with direction-preserving AUC and multiplicity adjustment; (ii)~non-inferiority
and equivalence tests for paired AUCs; (iii)~omnibus Wald test for three or
more correlated AUCs; (iv)~bootstrap rank-stability analysis; and (v)~Hurdle-AUC
for zero-inflated data.

For a single biomarker with smoothed ROC curves, \texttt{pROC} remains the
standard. For speed on $>10^5$ columns, \texttt{colAUC} is recommended. For
precision-recall of imbalanced data, \texttt{precrec} is preferred. For
combining two biomarkers, \texttt{dtComb} provides the most exhaustive methods.
For end-to-end biomarker screening with full inference, \texttt{aucmat} is the
recommended tool.

\section{Discussion}
% ===========================================================================

\texttt{aucmat} provides a unified, statistically principled framework for
biomarker screening. Its matrix-first design reduces the code burden from
dozens of scripting steps to a few function calls.

\textbf{Limitations.} The current version (0.2.0) is restricted to binary
outcomes for the primary workflow (multiclass is experimental). DeLong requires
common subjects; unpaired comparisons are not implemented (pROC provides this).
Partial AUC is an internal engine not yet exposed in \texttt{aucmat()}. The
DeLong estimator is slightly liberal at small $n$ (coverage $\approx 0.90$ at
$n=100$), consistent with its finite-sample behaviour; bootstrap intervals are
recommended when $n < 100$ per class.

\textbf{Future work.} v0.3: nested cross-validation for penalised panel
models. v0.4: multiclass AUC with one-versus-rest, one-versus-one, and
Hand-Till estimators. v0.5: cumulative/dynamic time-dependent AUCs for
survival outcomes \citep{Heagerty2000,Blanche2013}. v0.6: sparse-matrix
and on-disk backends for genomics-scale data.
